My Projects

OpenStreetMap Substation Viewer

This tool allows users to search for over 60,000 US substations by owner name (as included in the OpenStreetMap database). The tool also identifies substations that have been newly added to the OpenStreetMap within the last month.

USA Average Parcel Sizes

Average parcel sizes are important in prospecting for land development projects. I built a raster image representing average parcel size throughout most of the US by using Python and PostgreSQL to analyze over 150 million publicly available parcel centroid points.

Charlottesville and Albemarle Overland Flow Pathways

Understanding overland flow pathways and their drainage areas is useful in flood risk assessment, new development permitting, and identifying sources and destination for pollution. I built this web app in order to assist with these goals.
