Featured Project: Stavrocast

An Interactive Weather Forecast Tool

Stavrocast offers an intuitive way to access and visualize Global Ensemble Forecast System (GEFS) data for any airport with a valid ICAO code. Input an airport code to see a 24-hour meteogram, including temperature, dewpoint, wind, and precipitation probabilities, along with an embedded map for location context.

This tool leverages Python for backend data processing and provides a clean, responsive interface for easy interaction. Click below to try it out!
